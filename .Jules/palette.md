## 2025-05-22 - [Corporate UX Polish & Accessibility]
**Learning:** "Corporate perfect" UI requires extreme consistency in typography (using font-black for headings, font-bold for subheadings) and generous, purposeful spacing. Subtle backdrop blurs and shadow standards elevate the perceived value of the application. Focus-visible rings are essential for both accessibility and the "premium" feel.
**Action:** Always implement a global `focus-visible` style and use a strict hierarchy of font weights (black/bold/medium) to differentiate information layers.
